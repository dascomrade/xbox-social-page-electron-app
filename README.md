# build instructions
1. open project in your IDE (like VS Code)
2. run the build cmd associated with your OS
  Linux: npm run build:linux
  Windows: npm run build:win
  Scam (MacOS): npm run build:mac
3. go to the `dist` folder and run your AppImage/.snap/.EXE/.WhateverTheHellMacUses
